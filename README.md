# AnimationCounterSample

[ING] UILabelで数値が切り替わるアニメーション表示に関するサンプル

### 実装機能一覧

今回のサンプルに関しては、下記の2つの機能についてサンプルを実装しています。

+ Step1. 該当値に近づくにつれて変化のスピードが遅くなる
+ Step2. ドラムロール式の動きで値を変える
